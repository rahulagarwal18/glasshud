# 🌌 GlassHUD — Premium Cyberpunk Glassmorphism CSS Builder

GlassHUD is a premium, client-side web application that allows developers to visually configure glassmorphic user interface elements and instantly grab the generated CSS styles.

Designed to match the high-fidelity cyber-noir aesthetics of **GitPulse**, it features responsive sliders, real-time preview panels, and custom glowing elements.

---

## ✨ Features

- **Backdrop Blur Customizer**: Adjust the backdrop blur intensity up to 40px with real-time vector rendering.
- **Backdrop Opacity Controller**: Fine-tune card opacity to balance background visibility and text contrast.
- **Base Hue Slider**: Dynamically change card glow tones across the entire 360° color wheel.
- **Neon Glow Spread**: Create glowing cyberpunk borders and shadows with configurable spreads.
- **Border Radius Control**: Toggle between sharp corner grids or smooth rounded modern capsules.
- **One-Click CSS Exporter**: Instantly copy clean, structured CSS variables directly to your clipboard.

---

## 🛠️ Tech Stack

- **Markup**: HTML5 Semantic Structure
- **Styling**: Vanilla CSS3 (Custom properties, Noise overlays, responsive flex grids)
- **Logic**: Vanilla JavaScript (Interactive sliders, dynamic styling, clipboard integration)

---

## 🚀 Run Locally

1. Clone this repository.
2. Open `index.html` in your browser, or run a simple local web server:
   ```bash
   # Node.js
   npx serve .
   
   # Python
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` to start building glassmorphic UIs!
